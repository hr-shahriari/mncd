MNCD

Multi-layer Community Detection

MNCD is a .NET library written in C#, used for community detection in multi-layered networks.

This library implements multiple single and multi-layer algorithms for community detection and various methods for evaluating resulting communities.

Licensed under MIT, available on nuget.org.

Github (Source Code)

  • MNCD - .NET library.
  • MNCD APP - web aplication written in dotnet and react, used for analyzing multi-layer networks and communities within them.
  • MNCD VIZ - web api written in python for visualizing data in MNCD APP.
